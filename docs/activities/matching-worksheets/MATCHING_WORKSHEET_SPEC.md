# Matching Worksheet Family Spec

## Purpose

This document defines the high-level Matching Worksheet family direction.

## Objective

```text
Observation -> Matching -> Attribute Recognition -> Description -> Narration
```

## Current Supported Worksheet Types

- Exact Matching
- Attribute Matching

## Future Excluded Types

- Category Matching
- Function Matching
- Emotion Matching
- Multi-Attribute Matching
- AI Worksheet Generation

## UI Principles

- Large touch targets
- Single-card question model
- No drag-and-drop in V1
- No line drawing in V1
- ASD-friendly immediate feedback

## Reused NeuroBridge Systems

- Activity Shell
- Worksheet Shell
- Activity Completion
- Analytics
- Parent Dashboard
- Parent Administration Workspace
