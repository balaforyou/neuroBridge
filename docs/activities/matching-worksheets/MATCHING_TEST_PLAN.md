# Matching Worksheet Test Plan

This test plan defines focused coverage for Matching Worksheets.

## Required Tests

- Attribute worksheet loads
- Question card renders
- Three choices render
- Correct answer gives success feedback
- Incorrect answer gives gentle retry feedback
- Hint appears after repeated incorrect attempts when hints are ON
- Answer reveal occurs after max attempts
- Question count respects parent setting
- Attribute group selection controls dataset
- Completion screen appears
- Accuracy is calculated correctly
- Analytics entry is saved
- Parent Dashboard can read saved session
