# NB-WS-001.2 Attribute Matching Worksheet V1

Status:
FROZEN - Ready for Implementation

## Objective

```text
Observation -> Attribute Recognition -> Attribute Selection
```

## Scope Included

- Color
- Size
- State / Action
- Quantity
- Position

## Scope Excluded

- Category
- Function
- Emotion
- Multi-attribute
- AI-generated worksheets

## Question Format

- One visual/card prompt
- Three answer choices
- Single tap selection

Example:

```text
Prompt: Red Apple
Options: Red / Blue / Green
```

## Levels

### Level 1 - Color

Examples:

- Red Apple
- Blue Ball
- Green Leaf
- Yellow Banana

### Level 2 - Size

Examples:

- Big Ball
- Small Ball
- Tall Tree
- Short Plant

### Level 3 - State / Action

Examples:

- Sleeping Cat
- Running Dog
- Flying Bird

### Level 4 - Quantity

Examples:

- One Apple
- Three Apples
- Five Balls

### Level 5 - Position

Examples:

- Above
- Below
- Inside
- Outside

## Parent Controls

- Worksheet Type: Exact Matching / Attribute Matching
- Attribute Groups: Color, Size, State, Quantity, Position
- Question Count: 5, 10, 20
- Hints: ON / OFF
