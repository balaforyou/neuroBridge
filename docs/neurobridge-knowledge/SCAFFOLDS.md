## Scaffold Entries

### S-001: Color Cue Scaffold

- Scaffold ID: S-001
- Name: Color Cue Scaffold
- Type: Attribute Cue
- Level: 2
- Used In: Directions
- Learner Response: Helps connect an abstract direction to a concrete visual support.
- When To Use: When directional language is causing confusion.
- When To Fade: When learner responds accurately without color support.
- Risks: Learner may depend on color instead of direction.
- Notes: Useful bridge from abstract rule to visible cue.

### S-002: 3-2-1 Bottleneck Identification

- Scaffold ID: S-002
- Name: 3-2-1 Bottleneck Identification
- Type: Parent Prompt / Strategy Cue
- Level: 2
- Used In: Directions
- Learner Response: Helps learner pause and identify what matters before answering.
- When To Use: When the learner responds quickly but misses the relevant cue.
- When To Fade: When learner begins pausing independently.
- Risks: Parent prompt may become habitual if not faded.
- Notes: Good candidate for future self-talk scaffold.

### S-003: Soft Blink Cue

- Scaffold ID: S-003
- Name: Soft Blink Cue
- Type: Attention Cue
- Level: 1
- Used In: Neural Bonds / Kumon-style hints
- Learner Response: Guides attention without directly revealing the answer.
- When To Use: When learner overlooks the relevant region.
- When To Fade: When scanning improves.
- Risks: Excessive motion may distract.
- Notes: Keep blink gentle and brief.

### S-004: Choice Reduction

- Scaffold ID: S-004
- Name: Choice Reduction
- Type: Choice Reduction
- Level: 3
- Used In: Future frustration safety and assessment circuit breaker
- Learner Response: Reduces overwhelm when too many choices block response.
- When To Use: After repeated errors or frustration.
- When To Fade: When learner succeeds with reduced choices.
- Risks: May reduce assessment purity if used too early.
- Notes: Useful as a safety scaffold.

### S-005: Need Help Button

- Scaffold ID: S-005
- Name: Need Help Button
- Type: Learner-Initiated Scaffold
- Level: Variable
- Used In: Future shared activity shell
- Learner Response: Teaches help-seeking and self-advocacy.
- When To Use: Always available as a learner-safe support entry.
- When To Fade: Evolve label or reduce prominence as independence grows.
- Risks: Learner may overuse if it is too visually dominant.
- Notes: Can evolve from "Need Help?" to "?" later.

### S-006: Observation Time Control

- Scaffold ID: S-006
- Name: Observation Time Control
- Type: Timing Scaffold
- Level: 1 or 2
- Used In: Pattern Memory / Grid Vision
- Learner Response: Allows learner to observe before responding.
- When To Use: When errors appear due to rushed observation.
- When To Fade: Gradually reduce observation time as accuracy stabilizes.
- Risks: Too much time may reduce challenge.
- Notes: Should be parent-configurable.
