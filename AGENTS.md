Project: Siraash / NeuroBridge

Purpose:
Scaffold-driven learning platform for neurodivergent learners.

Architecture Principles:
- Reuse game engine
- Metrics first
- Scaffold discovery
- Adaptive difficulty
- Parent configurable

Every game must:
- Emit metrics
- Support hints
- Support scaffold fading
- Support reward engine
